CloudOptions 定义了 Feat Cloud 服务的配置信息。

## packages
**类型**: `string[]`

**默认值**: `null`

定义了 Feat Cloud 启动时加载的实例包路径，默认扫描 classpath 下的所有实例包。

## externalBeans
**类型**: `Map<String, Object>`

**默认值**: `null`

定义了 Feat Cloud 启动时加载的外部 Bean 配置。
